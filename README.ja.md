# MVSEP - 音楽分離ツール

音楽をボーカル、伴奏、ドrums，贝斯などのトラックの分離デスクトップクライアントです。

## 主な機能

- **ドラッグ＆ドロップ** - オーディオファイルをウィンドウにドロップするだけで処理開始
- **ワンクリック実行** - アップロード → 分離完了待ち → 自動ダウンロード、手動操作不要
- **タスク管理** - リアルタイム進捗表示、キャンセル、ダウンロード、削除に対応
- **多様なアルゴリズム** - 複数の分離アルゴリズムとモデルから選択可能
- **ダウンロード再開** - 中断したダウンロードを途中から再開可能
- **プロキシ対応** - システムプロキシ、手動プロキシ、無プロキシをサポート

## ダウンロードとインストール

### Windows

`MVSEP_x.x.x_x64.exe` または `MVSEP_x.x.x_x64.msi` をダウンロードしてインストール。

### Linux

AppImage または DEB パッケージをダウンロード：
- AppImage：実行権限を与えて直接実行
- DEB：ダブルクリックでインストールまたは `sudo dpkg -i MVSEP_xxx.deb`

### macOS

`.dmg` パッケージをダウンロードし、Applications フォルダにドラッグ。

## クイックスタート

### 1. 初回設定

初めて使う時に以下を設定：

| 設定項目 | 説明 |
|----------|------|
| **API Token** | 必須。[MVSEP](https://mvsep.com/user-api) で取得 |
| **出力ディレクトリ** | 分離結果の保存場所、デフォルト: `./output` |
| **出力形式** | MP3/WAV/FLAC/M4A から選択 |

### 2. 分離開始

1. **ホーム** でオーディオファイルをドラッグまたはクリックして選択
2. **アルゴリズム** と **モデルオプション** を選択（任意）
3. **出力形式** を選択
4. **ワンクリック実行** をクリック、完了后会自動ダウンロード

### 3. タスク確認

- **タスク** ページで進行中と過去のタスクを一覧表示
- **ダウンロード** で各ファイルを個別にダウンロード
- 進行中のタスクは **キャンセル** 可能

## よくある質問

### API Token の取得方法は？

1. [MVSEP](https://mvsep.com) にログイン
2. 右上のユーザー名をクリック → **API** を選択
3. Token をコピーしてクライアントの設定ページに貼り付け

### 分離が遅い場合の対処法は？

- **タスク** ページのキュー情報を確認して待ち人数を確認
- 別のアルゴリズムを試す（速いものももある）
- デモモードを検討（無料だが結果は公開される）

### ダウンロードが中断した場合は？

心配不要です。クライアントは**ダウンロード再開**をサポートしています。再度ダウンロードボタンをクリックすると途中から再開します。

### アルゴリズムリストを更新するには？

**アルゴリズム** ページで「最新アルゴリズム情報を取得」をクリックしてサーバーから最新を取得。

## ページ概要

| ページ | 機能 |
|--------|------|
| ホーム | オーディオアップロード、パラメータ選択、ワンクリック実行 |
| タスク | 進捗確認、ダウンロード結果、タスク管理 |
| アルゴリズム | アルゴリズムとモデルの閲覧、プリセット保存 |
| 設定 | API Token、プロキシ、出力ディレクトリなど |
| ログ | トラブルシューティング用ログ表示 |
